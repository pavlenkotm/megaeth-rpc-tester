## Additional Customization

### 1. Add Pinned Repositories
Go to your GitHub profile ‚Üí Customize your pins ‚Üí Select your best repositories

### 2. Enable GitHub Actions for Auto-Updates

Create `.github/workflows/blog-post-workflow.yml` to auto-update blog posts:

```yaml
name: Latest blog post workflow
on:
  schedule:
    - cron: '0 * * * *'
  workflow_dispatch:

jobs:
  update-readme-with-blog:
    name: Update this repo's README with latest blog posts
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: gautamkrishnar/blog-post-workflow@master
        with:
          feed_list: "https://your-blog.com/feed.xml"
```

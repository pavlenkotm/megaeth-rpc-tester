# Elm - Type-Safe Web3 Frontend

![Elm](https://img.shields.io/badge/Elm-blue)
![Frontend](https://img.shields.io/badge/platform-Web-green)

## Features
- ✅ Pure functional UI
- ✅ No runtime errors
- ✅ Web3 wallet integration
- ✅ Type-safe HTTP requests

## Installation
```bash
npm install -g elm
elm install elm/http elm/json
```

## Usage
```bash
elm make Main.elm --output=main.js
```

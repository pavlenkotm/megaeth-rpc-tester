Quick Start

1. Install MetaMask

Download and install the MetaMask browser extension from metamask.io

2. Connect Wallet

Click the "Connect Wallet" button and approve the connection in MetaMask

3. Get Test ETH

For testing, get free testnet ETH from a faucet (Sepolia, Goerli, etc.)

4. Start Using

Interact with smart contracts and explore decentralized features

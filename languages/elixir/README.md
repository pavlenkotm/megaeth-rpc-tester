# Elixir - Concurrent Web3 Backend

![Elixir](https://img.shields.io/badge/Elixir-purple)
![Concurrent](https://img.shields.io/badge/Concurrency-Actor_Model-green)

## Features
- ✅ GenServer-based RPC client
- ✅ Phoenix web framework integration
- ✅ OTP supervision trees
- ✅ Fault-tolerant blockchain nodes

## Installation
```bash
mix deps.get
mix compile
```

## Usage
```elixir
iex -S mix
Web3Client.get_block_number()
```

# OCaml - Type-Safe Web3

![OCaml](https://img.shields.io/badge/OCaml-orange)
![Functional](https://img.shields.io/badge/paradigm-Functional-blue)

## Features
- ✅ Strong static typing
- ✅ Pattern matching
- ✅ Lwt for async I/O
- ✅ JSON-RPC client

## Installation
```bash
opam install dune cohttp-lwt-unix yojson
dune build
```

## Usage
```bash
dune exec web3_client
```

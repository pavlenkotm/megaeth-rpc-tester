# Dart - Flutter Web3 Mobile DApps

![Dart](https://img.shields.io/badge/Dart-Flutter-blue)
![Platform](https://img.shields.io/badge/platform-Mobile-green)

## Overview

Dart powers Flutter, making it perfect for building cross-platform Web3 mobile applications, wallets, and DApp interfaces for iOS and Android.

## Features

- ✅ web3dart Ethereum Library
- ✅ Flutter UI Framework
- ✅ Cross-platform Mobile Apps
- ✅ WalletConnect Support

## Installation

```bash
# Install Dart SDK
brew install dart

# Or download from dart.dev
# Install Flutter
brew install flutter
```

## Usage

```bash
# Run Dart script
dart run web3_client.dart

# Flutter app
flutter pub get
flutter run
```

## Examples

### Web3 Client (`web3_client.dart`)

Ethereum client for Dart:
- Async operations
- Smart contract interaction
- Transaction building
- Event listening

### Flutter Wallet (`wallet_app.dart`)

Complete mobile wallet:
- Balance display
- Send/receive
- NFT viewer
- DApp browser

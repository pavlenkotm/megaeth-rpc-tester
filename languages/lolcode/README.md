## Syntax Highlights

### Basic Commands

```lolcode
HAI 1.2                    BTW Start program (version 1.2)
VISIBLE "Hello Web3"       BTW Print output
I HAS A var ITZ 42         BTW Declare variable
KTHXBYE                    BTW End program
```

### Variables

```lolcode
I HAS A balance ITZ 100           BTW Declare with value
I HAS A wallet                    BTW Declare without value
wallet R "0x123..."               BTW Assign value
balance R SUM OF balance AN 50    BTW Arithmetic
```

### Control Flow

```lolcode
O RLY?                    BTW If statement
  YA RLY                  BTW Then
    VISIBLE "Yes!"
  NO WAI                  BTW Else
    VISIBLE "No!"
OIC                       BTW End if

IM IN YR loop             BTW Loop
  BTW Loop body
IM OUTTA YR loop          BTW End loop
```

### Arithmetic

```lolcode
SUM OF a AN b             BTW Addition
DIFF OF a AN b            BTW Subtraction
PRODUKT OF a AN b         BTW Multiplication
QUOSHUNT OF a AN b        BTW Division
MOD OF a AN b             BTW Modulo
BIGGR OF a AN b           BTW Maximum
SMALLR OF a AN b          BTW Minimum
```

### Comparison

```lolcode
BOTH SAEM a AN b          BTW Equal
DIFFRINT a AN b           BTW Not equal
BIGGR OF a AN b           BTW Greater than
SMALLR OF a AN b          BTW Less than
```

## Usage

### Using lci (LOLCODE Interpreter)

```bash
# Install lci
git clone https://github.com/justinmeza/lci.git
cd lci
make
sudo make install

# Run LOLCODE programs
lci web3_wallet.lol
lci ethereum_rpc.lol
lci gas_calculator.lol
```

## Language Features

### Types

- **YARN**: String
- **NUMBR**: Integer
- **NUMBAR**: Float
- **TROOF**: Boolean (WIN/FAIL)
- **NOOB**: Untyped/null

### Keywords

- `HAI` - Start program
- `KTHXBYE` - End program
- `BTW` - Comment
- `OBTW`/`TLDR` - Multi-line comment
- `I HAS A` - Variable declaration
- `R` - Assignment
- `VISIBLE` - Print output
- `GIMMEH` - Input
- `O RLY?` - If statement
- `YA RLY` - Then
- `NO WAI` - Else
- `OIC` - End if
- `IM IN YR` - Loop start
- `IM OUTTA YR` - Loop end

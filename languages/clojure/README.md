# Clojure - Functional Web3 Development

![Clojure](https://img.shields.io/badge/Clojure-green)
![JVM](https://img.shields.io/badge/platform-JVM-orange)

## Features
- ✅ Immutable data structures
- ✅ Web3j integration
- ✅ Ring web server
- ✅ Async core.async channels

## Installation
```bash
lein deps
lein compile
```

## Usage
```clojure
lein repl
(require '[web3-client :refer :all])
(get-block-number)
```
